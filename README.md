# AVEVIBE 👕  
### Wear Your Vibe

A modern youth fashion brand website built as a lightweight single-page eCommerce interface for showcasing and managing apparel such as anime tees, jerseys, gym wear, oversized t-shirts, and custom designed shirts.

🌐 **Live Preview:** https://avevibe.netlify.app  

---

## 📌 Project Overview

AVEVIBE is a fashion brand concept focused on **affordable and stylish clothing for Indian youth and college students**.

This website serves as:

- Brand landing page
- Product showcase
- Catalog for clothing categories
- Custom t-shirt design concept
- Admin product management interface

The project is built using **pure HTML, CSS, and JavaScript** without heavy frameworks to keep it lightweight and easy to deploy.
